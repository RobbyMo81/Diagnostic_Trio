## Codebase Patterns
- Rust crate lives in `trio-host/`, Python package in `trio-reason/trio_reason/` (underscore, inside the hyphenated project dir)
- Python packaging uses `setuptools` with `setup.cfg` for `find:` package discovery
- Python venv at `/tmp/trio-venv` (or create fresh) — system Python is externally managed (PEP 668)
- Run Python checks: `mypy trio_reason tests` and `pytest` from inside `trio-reason/`
- Run Rust checks: `cargo test` from repo root (workspace)
